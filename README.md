# 出席確認システム 展示用
出席確認システムの展示用です。

## 必要なもの
* PC
* PaSoRi

## セットアップ
1. [このサイト](https://www.twilio.com/blog/an-easy-way-to-read-and-write-to-a-google-spreadsheet-in-python-jp)に従って鍵を生成し、ファイル名を `client_secret.json` にする
2. スプレッドシートを設定します
3. ライブラリインストール
   ```
   pip install -r requirements.txt
   ```
